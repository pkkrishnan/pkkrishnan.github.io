Supervised Machine Learning: CA Wildfire Prediction

Application of supervised machine learning models to predict the size of a wildfire (continuous variable) and a probability of a wildfire (classification). The project scope included: A literature review to understand the performance of external models and develop domain knowledge, Leveraged several techniques of ML in the project (Spatial join, PCA, Time series modeling, DNN, Sub-group analysis, and Automated Hyper Param Optimization), Built baseline shallow models (Linear Regression) to assess baseline metrics, Tuned DNN Model, Random Forest Regression, and Gradient Boost Regression to improve performance over baseline and external benchmarks and Developed a “Stretch Model (Prototype)” seeking to predict the rate of change in a burned area

Statistical Modelling: Rapid Transit Project Evaluation

Investing in rapid transit systems is very popular across the globe and such systems oftentimes drive economic value by driving urban growth. Rapid transit systems include subways, metros, and light rails. In this project, the application of statistical design, model building, and evaluation is demonstrated, to identify the key contributing features that impact the cost of a rapid transit project. Leveraging R, data exploration, data analysis, regression model, OLS assumption evaluation, and model selection were performed.
